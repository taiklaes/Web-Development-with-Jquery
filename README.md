# Web-Development with Jquery

Este repositório contém materiais, exercícios e projetos desenvolvidos durante o curso de extensão **Desenvolvedor Web - Biblioteca jQuery**, oferecido pelo IFSP (Instituto Federal de São Paulo).

## 📚 Sobre

O curso tem como objetivo apresentar os principais conceitos da Biblioteca jQuery, incluindo:

- Introdução ao jQuery
- Seletores
- Manipulação de DOM
- Eventos
- Efeitos e animações
- Ajax

## 📁 Estrutura

O repositório está organizado da seguinte forma:

```
/src
  └── /class-0X -> Pasta de cada aula
        └── index.html -> Código-fonte HTML
        └── script.js -> Código-fonte JavaScript
```

## ⚙️ Tecnologias

- HTML5
- CSS3
- JavaScript
- jQuery

## 🚀 Como usar?

1. Clone o repositório
2. Abra os arquivos `.html` no navegador
3. Utilize o DevTools para testes e inspeção

## 🎯 Objetivo

Consolidar o aprendizado em jQuery por meio da prática, servindo também como material de consulta futura.

## 📌 Observações

Este repositório é de caráter educacional e faz parte de um curso de extensão do IFSP.
